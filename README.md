# VirtualR

A modern, responsive landing page for **VirtualR** — a VR development platform that lets you build and deploy VR applications with ease.

🔗 **Live Demo:** [virtual-r-liart-alpha.vercel.app](https://virtual-r-liart-alpha.vercel.app/)

---

## 📋 Overview

VirtualR is a landing page built with React showcasing a fictional VR development tool. It includes sections for features, workflow, pricing, and testimonials.

## 🚀 Features

- Responsive navbar with smooth navigation
- Features section with icon cards
- Workflow / checklist section
- Pricing tiers (Free, Pro, Enterprise)
- Testimonials carousel
- Footer with resource, platform, and community links

## 🛠️ Tech Stack

- **React** — UI framework
- **Tailwind CSS** — Styling
- **Lucide React** — Icons
- **Vercel** — Deployment

## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/virtual-r.git

# Navigate into the project
cd virtual-r

# Install dependencies
npm install

# Start the development server
npm run dev
```
